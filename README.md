# API Routes
| Request Type | Endpoint URL                       | Description                                   |
|--------------|------------------------------------|-----------------------------------------------|
| PUT | `/api/v1/user/signup` | User signup |
| POST | `/api/v1/user/signin` | User signin |
| GET | `/api/v1/user/details` | User details |
| PUT | `/api/v1/blog` | Create a blog |
| POST | `/api/v1/blog` | Update a blog |
| GET | `/api/v1/blog/all` | Get all blogs |
| GET | `/api/v1/blog/:id` | Get a blog's details |

# Tech Used:
React.js, TailwindCSS, Hono, Cloudflare Workers
